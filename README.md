# Budget Tracker

A simple expense tracker for logging transactions, grouping spending by category, and seeing monthly totals.

## Features

- Add income or expense entries
- Categorize transactions
- Show balance, income, and expense totals
- Store data in localStorage
- Render category summaries

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Goals

This project practices form handling, array methods, browser storage, and clear financial summaries.
